# Juego El Ahorcado

## Descripción: 
Este es un juego web llamado "El Ahorcado" desarrollado con HTML, CSS y JavaScript. El objetivo del juego es adivinar una palabra oculta seleccionando letras.

## Tecnologías utilizadas:
  - HTML
  - CSS
  - JavaScript
