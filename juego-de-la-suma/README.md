# Juego de la Suma

## Descripción: 
Este es un juego web simple llamado "Juego de la Suma" creado con HTML, CSS y JavaScript. El juego presenta problemas de suma y desafía al jugador a resolverlos.

## Tecnologías utilizadas:
  - HTML
  - CSS
  - JavaScript
